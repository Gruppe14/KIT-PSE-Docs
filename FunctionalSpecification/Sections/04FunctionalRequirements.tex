\begin{enumerate}[resume]
  
  \item Provide access via web page with administrator rights
   
  \item Provide the oppertunity to initialize the data warehouse * % may get kicked out

\subsubsection*{Parser specific} 
 
 The following functions specify the parses functionality. Thereby /FXU/* to /FXZ/* specify
 the function /FXT/.
 
\begin{enumerate}[resume]
  
  \item Extract specific data from the log-files
  
  \item Extract access database, access time and user information (dimensions) % add to glossary: first
  
  \item Extract number of rows, elapsed time, busy time (measures) % add to glossary: all 3
  
  \item Extract type of data requested from the \texttt{where} part % add to glossary: where
  
  \item Transform the data to fit into the data warehouse schema
  
  \item Load the data into the data warehouse

\end{enumerate}
